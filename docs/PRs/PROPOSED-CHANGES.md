# Proposed Changes

Agent-generated proposals for updating `CONVENTIONS.md`, `QUESTIONS.md`, `PR-TEMPLATE.md`, or the validator. Nothing here is ever applied silently — a human must review and promote.

See `REFLECTION.md` for the loop.

## Schema
Each proposal is a fenced block:

````
### PROPOSAL-YYYY-MM-DD-<short-slug>
- **source:** script | agent-judgment
- **signal:** one-line description of what triggered this
- **target file:** CONVENTIONS.md | QUESTIONS.md | PR-TEMPLATE.md | regen-pr-index.mjs | REFLECTION.md
- **proposed change:** concrete diff or instruction
- **evidence:** pr_ids or file paths supporting this
- **confidence:** low | medium | high
- **needs_human:** true | false
- **status:** proposed | accepted | rejected
- **resolution:** (filled when accepted/rejected; one line)
````
